About Me

As a frontend web developer, I specialize in building responsive, interactive websites using HTML, CSS, and JavaScript, with a strong focus on GSAP for creating smooth animations. My expertise lies in translating designs into efficient, accessible code, ensuring seamless user experiences across all devices. I prioritize performance and usability, staying updated with the latest frontend trends to deliver cutting-edge web solutions.

I develop scalable, secure backend systems using Node.js and Express.js, with a strong focus on MongoDB for efficient data management and Passport.js for robust authentication. I create RESTful APIs to ensure smooth communication between server and frontend, emphasizing performance and security. My backend solutions are designed to handle large volumes of data and users reliably.

I create visually compelling posters and flex designs using Adobe Photoshop, Illustrator, and Canva. With a strong understanding of color theory, typography, and composition, I design impactful visuals that align with branding guidelines. My work is tailored to communicate effectively and leave a lasting impression, whether for events, promotions, or advertising.

I design user-centered, aesthetically pleasing interfaces with expertise in wireframing, prototyping, and user research. Using tools like Adobe XD, I create detailed, interactive prototypes focused on usability and accessibility. My designs prioritize intuitive user experiences, continuously refined through feedback and testing, ensuring high standards of usability and visual appeal.
